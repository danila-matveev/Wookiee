# Bitrix CRM Agent

## Бизнес-описание

**Назначение:** Интеграция с CRM-системой Bitrix24 для автоматизации работы с задачами, процессами и базой знаний компании Wookiee.

**Статус:** Планируется (OAuth-клиент и инфраструктура готовы, бизнес-логика не реализована)

**Какие задачи будет решать:**
- Постановка задач сотрудникам из Telegram-бота
- Анализ текущих процессов и загруженности команды
- Работа с базой знаний компании
- Синхронизация данных между Bitrix24 и аналитическими системами

**Кто будет использовать:** Руководство, менеджеры

---

## Технические детали

### Текущее состояние

Реализован OAuth 2.0 клиент для Bitrix24 REST API:

- Авторизация через OAuth 2.0
- Автоматическое обновление токенов
- Rate limiting (2 запроса/сек)
- Хранение токенов в файле
- Управление сессиями

### Архитектура

```
Telegram Bot / CLI
    ↓
Bitrix/client.py      → OAuth 2.0 клиент
    ↓
Bitrix24 REST API     → CRM, задачи, база знаний
```

### Ключевые файлы

| Файл | Назначение |
|------|-----------|
| `Bitrix/client.py` | OAuth 2.0 клиент с автообновлением токенов |
| `Bitrix/config.py` | Конфигурация (домен, client_id, secret) |
| `Bitrix/__init__.py` | Пакет |

### Конфигурация (.env)

```env
BITRIX_PORTAL_DOMAIN=...    # домен портала компании
BITRIX_CLIENT_ID=...        # ID приложения
BITRIX_CLIENT_SECRET=...    # секрет приложения
BITRIX_REDIRECT_URI=...     # URL для OAuth callback
```

### Поток авторизации

1. Получить URL авторизации → перейти в браузере
2. Пользователь разрешает доступ → Bitrix24 возвращает код
3. Обменять код на access_token и refresh_token
4. Делать API-запросы с access_token
5. При истечении → автоматический refresh

## Планы реализации

### Фаза 1: Базовая интеграция
- Чтение задач из Bitrix24
- Поиск по базе знаний
- Отображение в Telegram-боте

### Фаза 2: Автоматизация
- Создание задач из бота
- Привязка аналитических данных к задачам
- Автоматические отчёты по статусу задач

### Фаза 3: Аналитика процессов
- Анализ загруженности команды
- Выявление узких мест в процессах
- Рекомендации по оптимизации

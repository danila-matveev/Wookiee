# Definition of Done (DoD)

Задача считается завершённой когда ВСЕ пункты выполнены:

## Код

- [ ] Код работает по требованиям (протестировано)
- [ ] Нет хардкоженных секретов или кредов
- [ ] GROUP BY по модели использует LOWER() где применимо
- [ ] Процентные метрики используют средневзвешенные для комбинированных каналов
- [ ] Оба значения (current и previous) вычисляются для сравнений

## Документация

- [ ] README.md обновлён если изменились скрипты/CLI
- [ ] AGENTS.md обновлён если изменились правила
- [ ] docs/development-history.md обновлён
- [ ] ADR создан если принято архитектурное решение
- [ ] docs/agents/README.md обновлён если добавлен/изменён агент
- [ ] docs/architecture.md обновлён если изменена архитектура
- [ ] Все ссылки в обновлённых .md файлах валидны

## Качество данных

- [ ] Чеклист из DATA_QUALITY_NOTES.md соблюдён
- [ ] Новые проблемы данных задокументированы в DATA_QUALITY_NOTES.md

## Безопасность

- [ ] Нет кредов в коммитируемых файлах
- [ ] .env.example содержит только плейсхолдеры

## По типу задачи

### Новая функция
- [ ] Описание добавлено в README
- [ ] Примеры использования в README

### Баг-фикс
- [ ] Корневая причина идентифицирована
- [ ] Регрессионный тест добавлен (или задокументировано почему нет)

### Рефакторинг
- [ ] Функциональность не изменилась
- [ ] ADR создан если рефакторинг значительный

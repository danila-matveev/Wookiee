# Sports Pre-Launch Research — Design Spec

> Разовый глубокий анализ рынка спортивной женской одежды и аксессуаров на WB для принятия решения о запуске бренда TELOWAY.

**Дата:** 2026-04-07
**Формат:** Команда субагентов → реальные данные MPStats API → Notion-страница

---

## Цель

Собрать и проанализировать данные по рынку женской спортивной одежды на WB:
- Объёмы и динамика категорий
- Конкуренты и их драйверы роста
- Ценовые сегменты (фокус: средний-премиум)
- Лучшие SKU и новинки
- Точки входа для TELOWAY
- Аксессуары (сумки, коврики, носки)

Результат: одна подробная Notion-страница с реальными данными из MPStats.

---

## Архитектура

```
scripts/sports_research/run.py
  │
  ├─ Stage 1: Parallel Data Collection (субагенты)
  │    ├─ Agent 1: Категории (15+ категорий, март vs фев, объёмы, ср.чек)
  │    ├─ Agent 2: Конкуренты из карты (12 брендов, top SKU, subject breakdown)
  │    ├─ Agent 3: Discovery (топ-бренды в каждой категории вне нашего списка)
  │    ├─ Agent 4: Лучшие SKU рынка (топ по выручке в каждой категории)
  │    └─ Agent 5: Новинки (sku_first_date >= 2025-10-01, с высокой выручкой)
  │    --> /tmp/sports_research_data.json
  │
  ├─ Stage 2: Верификация (кросс-проверка данных)
  │
  ├─ Stage 3: LLM-аналитик (HEAVY) → генерация отчёта
  │
  └─ Stage 4: Публикация в Notion
```

## Категории для анализа

### Основная одежда
| Категория | Предполагаемый MPStats path | Примечание |
|-----------|---------------------------|------------|
| Леггинсы/тайтсы | Женщинам/Одежда/Леггинсы + Спортивная одежда/Леггинсы | Ключевая категория |
| Спортивные шорты | Спортивная одежда/Шорты спортивные | |
| Спортивные топы/бра | Спортивная одежда/Спортивные бюстгальтеры + Топы | |
| Рашгарды | Спортивная одежда/Рашгарды | |
| Спортивные костюмы | Спортивная одежда/Костюмы спортивные | |
| Ветровки/куртки | Спортивная одежда/Куртки спортивные | Для бега |
| Юбки спортивные | Спортивная одежда/Юбки | Падел/теннис |
| Комбинезоны | Спортивная одежда/Комбинезоны | Йога |

### По направлениям (для анализа позиционирования)
- **Йога/пилатес** — леггинсы + топы + комбинезоны (мягкие ткани, приглушённые цвета)
- **Фитнес/зал** — леггинсы + шорты + топы (компрессия, яркие цвета)
- **Бег** — тайтсы + шорты + ветровки (лёгкие ткани, светоотражатели)
- **Падел/теннис** — юбки + топы (специфичная ниша, растущая)

### Аксессуары
| Категория | MPStats path (предполагаемый) |
|-----------|------------------------------|
| Спортивные сумки | Спорт/Сумки спортивные |
| Коврики для йоги | Спорт/Коврики для йоги |
| Носки для пилатеса | Спорт/Носки спортивные |

**Важно:** Точные MPStats paths определяются при реализации — скрипт сначала проверяет доступные пути через API.

## Конкуренты

### Из карты (12 брендов)
IRNBY, MELA (Sport Mela), Profit Wear, Euphoria, Bona Fide, EAZYWAY, Lavarice, Fluide, Forward, Zasport, Demix, Sport Angel

Для каждого:
- Выручка март vs февраль, рост %
- Subject breakdown (за счёт каких категорий выручка)
- Топ-5 SKU по выручке (название, цена, продажи, рейтинг, дата запуска)
- Новые запуски (sku_first_date >= 2025-10-01)
- Вывод: что копировать/адаптировать

### Discovery (автоматический поиск)
В каждой категории:
- Получаем топ-10 брендов по выручке
- Фильтруем тех, кого нет в списке конкурентов
- Для каждого нового бренда: выручка, top SKU, позиционирование

## Данные, которые собираем

### По каждой категории (март 2026 vs февраль 2026)
- `total` — количество товаров в категории
- `sample_revenue` — выручка топ-500 товаров (sample)
- `sample_sales` — продажи топ-500
- `avg_price` — средний чек
- Топ-10 брендов в категории (с выручкой)
- Топ-20 SKU по выручке (имя, бренд, цена, продажи, рейтинг, дата запуска)
- Новинки: товары с sku_first_date >= 2025-10-01 и revenue > 500K

### По каждому конкуренту
- `get_brand_trends` → top-100 SKU с revenue, sales, subject, price, etc.
- Март vs февраль для расчёта роста
- Subject breakdown из поля `subject` каждого SKU

### Ценовые сегменты
Из топ-SKU каждой категории группируем по цене:
- Эконом: до 3 000 ₽
- Средний: 3 000–6 000 ₽
- Премиум: 6 000+ ₽
Считаем долю выручки в каждом сегменте.

## Структура Notion-отчёта

### I. Главные выводы
5 ключевых находок + рекомендация по запуску (на основе данных, не допущений)

### II. Обзор рынка по категориям
Таблица: категория → объём (выручка sample) → рост MoM → товаров → ср.чек
- Какие категории самые большие
- Какие растут быстрее всех
- Где больше всего конкуренции (товаров/продавцов)

### III. Ценовые сегменты
По ключевым категориям (леггинсы, топы, шорты):
- Распределение выручки по сегментам (эконом/средний/премиум)
- Где концентрация выручки
- Топ-5 SKU в среднем-премиум сегменте (наш целевой)

### IV. Конкуренты: глубокий анализ
По каждому из 12 брендов:
- Toggle: бренд → выручка → рост
- Внутри: subject breakdown + top-5 SKU + новые запуски + вывод для TELOWAY

Discovery: новые растущие бренды вне списка

### V. Лучшие SKU рынка
Топ-30 товаров по выручке во всех спортивных категориях:
- Таблица: бренд | название | категория | выручка | продажи | цена | рейтинг | дата запуска

### VI. Новинки-взрывы
Товары, запущенные с Q4 2025, с высокой выручкой:
- Что выстрелило за последние 6 месяцев
- Какие форматы/типы товаров

### VII. Точки входа для TELOWAY
На основе данных:
- Какие категории запускать первыми (объём + рост + конкуренция)
- Оптимальный ценовой сегмент
- Какие конкретные типы товаров (модели/фасоны) показывают лучшие результаты
- Рекомендация по первым 10-15 SKU

### VIII. Аксессуары
Отдельная секция:
- Сумки спортивные — объём, конкуренция, стоит ли
- Коврики для йоги — объём, конкуренция
- Носки для пилатеса — объём, конкуренция

## Реализация

### Переиспользуемые компоненты
- `shared/clients/mpstats_client.py` — уже есть
- `shared/config.py` — MPSTATS_API_TOKEN уже есть

### Новые файлы
| Файл | Назначение |
|------|------------|
| `scripts/sports_research/__init__.py` | Package |
| `scripts/sports_research/config.py` | Категории, конкуренты, пути MPStats |
| `scripts/sports_research/collectors/categories.py` | Сбор данных по категориям |
| `scripts/sports_research/collectors/competitors.py` | Данные по 12 конкурентам + discovery |
| `scripts/sports_research/collectors/top_skus.py` | Лучшие SKU в каждой категории |
| `scripts/sports_research/collectors/new_items.py` | Новинки (Q4 2025+) |
| `scripts/sports_research/collectors/accessories.py` | Аксессуары |
| `scripts/sports_research/collect_all.py` | Оркестратор (параллельный сбор) |
| `scripts/sports_research/prompts/analyst.md` | Промпт для LLM-аналитика |

### Публикация
Notion-страница рядом с конкурентной картой:
- Parent: `33a58a2bd58781b09a61cbef80df4922` (Аналитика конкурентов — Белье + Спорт)
- Формат: как market-review — `<table>`, `<callout>`, toggle headings, цвета

### Запуск
```bash
python3 scripts/sports_research/collect_all.py --output /tmp/sports_research.json
```
Затем LLM-аналитик обрабатывает JSON и публикует в Notion.

## Button

- Exports: `Button` (forwardRef component), `ButtonProps` (type)
- Props:
  ```ts
  interface ButtonProps extends ButtonHTMLAttributes<HTMLButtonElement> {
    variant?: 'primary' | 'secondary' | 'ghost' | 'danger'; // default: 'secondary'
    loading?: boolean;
  }
  ```
- Usage: `<Button variant="primary" onClick={handleSave}>Сохранить</Button>`
- Notes: `loading` shows an animated spinner and sets `disabled`. Min height 36px. Extends all native `<button>` HTML attributes.

## Badge

- Exports: `Badge`, `BadgeProps` (type)
- Props:
  ```ts
  interface BadgeProps {
    tone?: 'success' | 'warning' | 'info' | 'orange' | 'pink' | 'secondary' | 'danger'; // default: 'secondary'
    children: ReactNode;
    className?: string;
  }
  ```
- Usage: `<Badge tone="success">Активен</Badge>`
- Notes: 7 tones total. `orange` maps to `bg-primary-light text-primary-hover`. `secondary` has border. Rendered as `<span>` with `rounded-full` pill shape, 11px font.

## Drawer

- Exports: `Drawer`
- Props:
  ```ts
  interface DrawerProps {
    open: boolean;
    onClose: () => void;          // NOTE: onClose, NOT onOpenChange
    title: string;
    children: ReactNode;
    footer?: ReactNode;
    width?: string;               // Tailwind class, default: 'max-w-2xl'
  }
  ```
- Usage: `<Drawer open={open} onClose={() => setOpen(false)} title="Редактировать">...</Drawer>`
- Notes: Built on `@radix-ui/react-dialog`. Slides in from the **right**. Accepts optional `footer` for action buttons. Width is a Tailwind `max-w-*` class string. **IMPORTANT: prop is `onClose`, not `onOpenChange`** — differs from plan template.

## EmptyState

- Exports: `EmptyState`, `EmptyStateProps` (type)
- Props:
  ```ts
  interface EmptyStateProps {
    title: string;
    description?: ReactNode;
    icon?: ReactNode;
    action?: ReactNode;
    className?: string;
  }
  ```
- Usage: `<EmptyState title="Нет данных" description="Добавьте первую запись." icon={<Inbox size={32} />} />`
- Notes: Renders a dashed-border card centered layout. `action` slot for a CTA button. Fully composable icon slot (pass any ReactNode, e.g. lucide icon).

## QueryStatusBoundary

- Exports: `QueryStatusBoundary` (named + default)
- Props:
  ```ts
  interface Props {
    isLoading: boolean;
    error: unknown;
    isEmpty?: boolean;
    loadingFallback?: ReactNode;
    emptyTitle?: string;           // default: 'Пусто'
    emptyDescription?: string;     // default: 'Снимите фильтры или добавьте первую запись.'
    children: ReactNode;
  }
  ```
- Usage: `<QueryStatusBoundary isLoading={isLoading} error={error} isEmpty={items.length === 0}>{children}</QueryStatusBoundary>`
- Notes: Priority order: `isLoading` → `error` → `isEmpty` → `children`. Error renders `role="alert"` div with danger styling. Uses `Skeleton` internally. Replaces ad-hoc loading ladders.

## PageHeader

- Exports: `PageHeader`, `PageHeaderProps` (type)
- Props:
  ```ts
  interface PageHeaderProps {
    title: string;
    sub?: string;
    actions?: ReactNode;
    className?: string;
  }
  ```
- Usage: `<PageHeader title="Маркетинг" sub="Отслеживайте кампании и блогеров" actions={<Button>Добавить</Button>} />`
- Notes: Renders `<header>` with `<h1>` using `font-display text-3xl font-bold`. Responsive: stacks vertically on mobile, row on `sm:`. `actions` slot aligns to the right.

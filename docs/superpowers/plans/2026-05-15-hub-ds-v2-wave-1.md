# Hub DS v2 Wave 1 — Foundation Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Перевести Wookiee Hub (`hub.os.wookiee.shop`) с трёх UI-диалектов (purple/dark Hub + catalog stone scope + marketing stone scope + CRM kit) на единый Design System v2 — stone + DM Sans + Instrument Serif + light/dark — Wave 1 (Foundation: tokens, fonts, primitives, layout shell, pages migration, QA).

**Architecture:**
- Token strategy «repaint values, keep names» — все CSS-vars из shadcn (`--background`, `--foreground`, `--primary` и т.д.) сохраняют имена, меняются значения с OKLCH-purple на stone. Это даёт каскад без рефакторинга 1000+ classes.
- Wookiee-extended токены (`--bg-soft`, `--text-dim`, `--icon-bar-bg`, `--bg-hover`, `--wk-green/red/yellow/blue/pink`) пересчитываются в обеих темах.
- Theme orchestration переносится в `main.tsx` (через `useThemeStore.subscribe`) + inline FOWT-script в `index.html`, чтобы `/login` (вне AppShell) тоже синхронизировался без flash.
- 11 новых primitives добавляются в `src/components/ui/*` строго по сигнатурам эталонного `wookiee_ds_v2_foundation.jsx`.
- Catalog (22 страницы) и Marketing (2 страницы) НЕ трогаются — только Playwright QA.
- CRM Kit (`src/components/crm/*`) и `/agents/*` — out of scope Wave 1.

**Tech Stack:**
- Vite 7 + React 19 + React Router 7
- Tailwind CSS v4 (`@theme inline`) + shadcn/ui + Radix UI
- Zustand 5 (persist), Sonner (toast), @dnd-kit, recharts
- DM Sans Variable + Instrument Serif (Google Fonts)
- Vitest для unit-тестов primitives; Playwright MCP для E2E QA
- Branch: `feat/ds-v2-wave-1-spec` (созданная отдельно чтобы не пересекаться с параллельной telemost-работой)

**Spec:** `docs/superpowers/specs/2026-05-15-hub-ds-v2-wave-1-design.md` (commit `241c5022`)

// wookiee-hub/src/components/ui/ring.tsx
import { cn } from "@/lib/utils"

export interface RingProps {
  value: number
  max?: number
  size?: number
  strokeWidth?: number
  showLabel?: boolean
  className?: string
}

export function Ring({
  value,
  max = 100,
  size = 48,
  strokeWidth = 4,
  showLabel = true,
  className,
}: RingProps) {
  const pct = Math.min(100, Math.max(0, (value / max) * 100))
  const r = (size - strokeWidth) / 2
  const c = 2 * Math.PI * r
  const offset = c - (pct / 100) * c
  return (
    <div className={cn("relative inline-flex items-center justify-center", className)} style={{ width: size, height: size }}>
      <svg width={size} height={size} className="-rotate-90">
        <circle cx={size / 2} cy={size / 2} r={r} stroke="currentColor" strokeWidth={strokeWidth} fill="none" className="text-muted opacity-30" />
        <circle cx={size / 2} cy={size / 2} r={r} stroke="currentColor" strokeWidth={strokeWidth} fill="none" className="text-foreground" strokeDasharray={c} strokeDashoffset={offset} strokeLinecap="round" />
      </svg>
      {showLabel && <span className="absolute text-xs font-medium">{Math.round(pct)}%</span>}
    </div>
  )
}

    <QueryClientProvider client={queryClient}>
      <RouterProvider router={router} />
      <Toaster richColors position="top-right" closeButton theme={theme} />
    </QueryClientProvider>
  )
}

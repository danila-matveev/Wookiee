# /bitrix-task — постановка задач в Битрикс24

**Запуск:** `/bitrix-task` (или фразы из триггеров: «поставь задачу», «задача для [имя]», «поручение», «сформулируй задачу», «надиктую задачу»).

## Что делает

Помогает сформулировать задачу по правилам компании и создаёт её в Битрикс24 через REST API после подтверждения пользователя.

Двухфазный подход для экономии Opus-токенов:
1. **Фаза A (Sonnet):** парсит запрос, готовит превью-карточку, ищет недостающие поля, готовит JSON для API.
2. **Фаза B (Opus):** показывает превью, принимает подтверждение/правки, выполняет API-вызов.

Включает справочник команды (ID и должности), маппинг полей Битрикс24, дефолты по дедлайнам, чек-листам и наблюдателям.

## Зависимости

- `shared.tool_logger.ToolLogger` — для телеметрии запусков (Supabase).
- Bitrix24 webhook URL в `.env` (`Bitrix_rest_api`).

## Запуск из корня репо

Tool Logging-блок использует `PYTHONPATH=.` — скилл предполагает запуск из корня `Projects/Wookiee/`.

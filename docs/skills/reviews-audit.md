# /reviews-audit — Аудит отзывов

**Запуск:** `/reviews-audit`

## Что делает

Анализирует отзывы покупателей по категориям проблем: качество, размерная сетка, упаковка, доставка. Выявляет системные претензии по моделям и динамику рейтинга.

## Параметры

- `model` — модель для фокусного анализа (опционально, по умолчанию: все)
- `period` — период (по умолчанию: последние 30 дней)
- `channel` — канал: `wb`, `ozon`, `all` (по умолчанию: `all`)

## Результат

Отчёт с категоризированными проблемами выводится в stdout.

## Зависимости

- Wildberries API (MCP `wildberries-ip`, `wildberries-ooo`)
- OZON API (`OZON_API_KEY`)

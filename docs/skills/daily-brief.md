# /daily-brief — Ежедневный брифинг

**Запуск:** `/daily-brief`

## Что делает

Формирует ежедневный брифинг за вчерашний день: ключевые метрики (выручка, маржа, заказы), топ-3 движения по моделям, флаги аномалий. Публикует в Notion.

## Параметры

- `date` — дата отчёта (по умолчанию: вчера, формат `YYYY-MM-DD`)

## Результат

Страница публикуется в Notion (база данных ежедневных брифингов).

## Зависимости

- PostgreSQL (DB Server, read-only)
- Notion API (`NOTION_TOKEN`, `NOTION_DAILY_BRIEF_DB_ID`)
- `shared/data_layer.py`

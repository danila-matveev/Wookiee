# /funnel-report — Воронка по моделям

**Запуск:** `/funnel-report`

## Что делает

Строит воронку конверсии по каждой модели: показы → клики → добавления в корзину → заказы. Рассчитывает CRO (конверсию) и halo-эффект (влияние модели на продажи связанных SKU).

## Параметры

- `period` — период (по умолчанию: последние 30 дней)
- `channel` — канал: `wb`, `ozon`, `all` (по умолчанию: `all`)

## Результат

Таблицы с CRO по моделям выводятся в stdout и опционально публикуются в Notion.

## Зависимости

- PostgreSQL (DB Server, read-only)
- `shared/data_layer.py`

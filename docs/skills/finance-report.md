# /finance-report — Финансовый отчёт

**Запуск:** `/finance-report`

## Что делает

Формирует финансовый отчёт по каналам WB+OZON: выручка, маржа, ДРР, логистика, расходы. Поддерживает декомпозицию по 5 рычагам маржи и сравнение с предыдущим периодом.

## Параметры

- `period` — период (по умолчанию: вчера)
- `channel` — канал: `wb`, `ozon`, `all` (по умолчанию: `all`)

## Результат

Отчёт публикуется в Notion (страница финансового отчёта) и выводится в stdout.

## Зависимости

- PostgreSQL (DB Server, read-only)
- Notion API (`NOTION_TOKEN`, `NOTION_FINANCE_PAGE_ID`)
- `shared/data_layer.py`

# /market-review — Обзор рынка

**Запуск:** `/market-review`

## Что делает

Формирует обзор рынка на основе данных MPStats: топ конкурентов по нишам, динамика продаж категорий, доля рынка бренда Wookiee. Выявляет растущие и падающие ниши.

## Параметры

- `category` — категория товаров для анализа
- `period` — период (по умолчанию: последние 30 дней)

## Результат

Отчёт публикуется в Notion и выводится в stdout.

## Зависимости

- MPStats API (`MPSTATS_API_KEY`)
- Notion API (`NOTION_TOKEN`)
- `shared/clients/mpstats_client.py`

## Critical Findings

### 1. 15 моделей из Sheets отсутствуют в Supabase

**Agent:** Schema Matcher | **Severity:** CRITICAL

Модели из "Все товары" (col F), которых нет в `modeli`:

`ashley, billie, carol, diana, emma, jackie, kylie, luna, nora, polly, rose, sabrina, sky, tina, другие товары`

Это тянет за собой:
- **70 артикулов** отсутствуют в `artikuly`
- **232 баркода** отсутствуют в `tovary`
- Некоторые баркоды (`11111`, `11112`, ...) — плейсхолдеры, не реальные

**Причина:** Sheets обновляется быстрее, чем Supabase. Новые модели добавлены в Sheets, но ещё не загружены в БД.

### 2. Расхождение modeli_osnova: "Set Bella" vs "Evelyn"

**Agent:** Schema Matcher | **Severity:** CRITICAL

- `set bella` — есть в Sheets (col H, Модель основа), нет в Supabase
- `evelyn` — есть в Supabase, нет в Sheets

Количество совпадает (23 = 23), но состав отличается. Возможно переименование или замена.

### 3. Сломанные FK: 4 артикула Angelina без цвета

**Agent:** Hierarchy Checker | **Severity:** CRITICAL

4 записи в `artikuly` ссылаются на несуществующие записи в `cveta`:

| id | artikul |
|----|---------|
| 451 | Angelina/white |
| 452 | Angelina/black |
| 453 | Angelina/dark_red |
| 454 | Angelina/nude |

Все 4 — модель Angelina. Цвета `white`, `black`, `dark_red`, `nude` отсутствуют в таблице `cveta`.

### 4. МойСклад ↔ Supabase: совпадение только 53% баркодов

**Agent:** MoySklad Validator | **Severity:** CRITICAL

| Метрика | Значение |
|---------|----------|
| МойСклад items (с баркодами) | 1 490 |
| Supabase tovary | 1 450 |
| Совпавшие баркоды | 795 (53%) |
| Только в МойСклад | 695 |
| Только в Supabase | 655 |

**Корневая причина:** Системы используют разные стандарты баркодов:
- МойСклад: внутренние коды формата `2000000XXXXXX`
- Supabase: WB-маркетплейсные баркоды формата `2000989...`, `2010165...`

Один и тот же физический товар может иметь разные баркоды в разных системах.

**Дополнительно:** 10 баркодов в Supabase с подозрительно короткими значениями (`10`, `18`, `7`) — ошибки ввода данных.

**Невозможна сверка артикулов:** Все 1 515 записей МойСклад имеют пустое поле `article`. Кросс-валидация по артикулу невозможна.

# Задачи

## 1. Матрица ABC × ROI (только "Продается")
- Сводная таблица 3×3: количество артикулов, сумма маржи в каждой ячейке
- Распределение: какие ячейки перегружены, какие пусты
- Топ-артикулы в каждой ячейке

## 2. Юнит-экономика топ-моделей
По топ-10 артикулам по выручке:
- Выручка → маржа → маржинальность%
- Структура расходов: доли ДРР, комиссии, логистики в выручке (что доступно из данных)
- Сравнение с медианой категории

## 3. Ценовые аномалии
- Маржинальность <15% у артикулов со статусом "Продается" → флаг ⚠️
- Высокая выручка + низкая маржа = неэффективное ценообразование

## 4. Выкуп % (информационно)
- Низкий выкуп (<70%) → повышенная обратная логистика → занижение реальной маржи
- Причины: размерная сетка, качество фото, несоответствие ожиданиям
- Пометка: "лаг 3-21 дн"

## 5. ИП vs ООО
Одна и та же модель (Vuki vs Vuki2, Moon vs Moon2):
- **РАЗНЫЕ стратегические цели:**
  - ИП = высокий объём + приемлемая маржа (массовый сегмент)
  - ООО = меньший объём + высокая маржа + лучшая упаковка (премиум)
- Оценка: достигает ли каждое юрлицо СВОЕЙ цели
- НЕ сравнивать "кто эффективнее" в абсолютных цифрах

## 6. "Что если" сценарии
Для артикулов с аномалиями:
- Если поднять/снизить цену на 5-10% → эффект на маржу (руб/мес)
- Если сократить стоки C-товаров → сколько капитала высвободится

## 7. Замороженный капитал
- Сколько денег заморожено в стоках C-товаров с низким ROI?
- Стоимость хранения этих товаров в месяц (если есть storage данные)

## 8. Товары "Выводим" — отдельная матрица
- ABC × ROI для артикулов в статусе "Выводим"
- Поиск кандидатов на возврат в "Продается" — хорошая маржинальность + приемлемая оборачиваемость

# Правила
- Причинно-следственные цепочки обязательны

- % метрики при объединении каналов — СРЕДНЕВЗВЕШЕННЫЕ

# agents/

Scaffold для будущих AI-агентов Фазы 2.

В текущем активном контуре (Фаза 1) логика автоматизации реализована через:
- **Скиллы** (`scripts/*/`) — CLI-инструменты, запускаемые из Claude Code
- **Сервисы** (`services/*/`) — долгоживущие процессы и API

Oleg V2 (финансовый агент) был выведен из эксплуатации в апреле 2026.
Архитектурные заметки: `docs/archive/oleg-v2-architecture.md`.

## ВАЖНО: Особенности Локализации отчёта

Этот шаблон — **data-driven** (без глубокой LLM аналитики). Это намеренное решение.

**Данные:**
- Источник: WB Marketplace API (локализация)
- Сервис: services/wb_localization/ (отдельный скрипт)
- Агент: Vasily отвечает за оптимизацию логистики и локализации

**НЕ включать:**
- Глубокий LLM-анализ
- Финансовые KPI (маржа, ДРР) — это отдельный финансовый отчёт

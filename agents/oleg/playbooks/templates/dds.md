## ВАЖНО: Особенности ДДС отчёта

Этот шаблон — **data-driven** (без глубокой LLM аналитики). Это намеренное решение.

**Данные:**
- Источник: Finolog API
- Формат: движение денег по статьям (поступления, расходы, остатки)
- Сервис: finolog-cron (отдельный скрипт)

**НЕ включать:**
- Глубокий LLM-анализ (не входит в задачу ДДС отчёта)
- Маркетинговые метрики (воронка, реклама)
- Финансовые KPI маркетплейсов (маржа, ДРР — это отдельный финансовый отчёт)

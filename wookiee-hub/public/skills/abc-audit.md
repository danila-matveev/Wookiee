# /abc-audit — ABC-аудит ассортимента

**Запуск:** `/abc-audit`

## Что делает

Выполняет ABC-классификацию ассортимента по маржинальной прибыли: A (~70%), B (~20%), C (~10%). Анализирует WB+OZON раздельно и в совокупности, помесячная динамика. Выявляет модели, теряющие позицию в классификации.

## Параметры

- `months` — количество месяцев для анализа (по умолчанию: 3)
- `channel` — канал: `wb`, `ozon`, `all` (по умолчанию: `all`)

## Результат

Таблица ABC-классификации выводится в stdout. Опционально публикуется в Notion.

## Зависимости

- PostgreSQL (DB Server, read-only)
- `shared/data_layer.py`

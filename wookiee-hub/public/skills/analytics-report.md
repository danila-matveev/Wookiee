# /analytics-report — Аналитический отчёт

**Запуск:** `/analytics-report`

## Что делает

Формирует сводный аналитический отчёт: трафик (показы, клики), заказы (штуки и рубли), продажи, конверсия по каналам WB+OZON. Сравнивает с предыдущим периодом.

## Параметры

- `period` — период (по умолчанию: последние 7 дней)
- `channel` — канал: `wb`, `ozon`, `all` (по умолчанию: `all`)
- `breakdown` — разбивка: `model`, `category`, `none` (по умолчанию: `model`)

## Результат

Отчёт выводится в stdout. Опционально публикуется в Notion.

## Зависимости

- PostgreSQL (DB Server, read-only)
- `shared/data_layer.py`

# /logistics-report — Отчёт по логистике WB

**Запуск:** `/logistics-report`

## Что делает

Рассчитывает переплаты Wildberries за логистику: сравнивает фактические тарифы с нормативными, выявляет аномалии по складам и моделям. Использует данные из таблицы `wb_tariffs` в Supabase.

## Параметры

- `period` — период анализа (по умолчанию: текущий месяц)
- `cabinet` — кабинет WB: `OOO`, `IP` (по умолчанию: оба)

## Результат

Отчёт с переплатами по складам выводится в stdout. Критические аномалии логируются.

## Зависимости

- PostgreSQL (DB Server, read-only)
- Supabase (`wb_tariffs` таблица)
- `services/logistics_audit/`
- `shared/data_layer.py`

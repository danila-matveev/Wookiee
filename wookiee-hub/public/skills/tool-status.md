# /tool-status — Статус инструментов

**Запуск:** `/tool-status`

## Что делает

Показывает статус последних запусков инструментов (скиллов): когда запускались, результат (success/error), длительность. Данные берутся из таблицы `tool_runs` в Supabase.

## Параметры

- `tool` — имя инструмента для фильтрации (опционально, по умолчанию: все)
- `last` — количество последних записей (по умолчанию: 10)

## Результат

Таблица статусов выводится в stdout.

## Зависимости

- Supabase (`tool_runs` таблица)
- `services/tool_telemetry/`
- `shared/tool_logger.py`

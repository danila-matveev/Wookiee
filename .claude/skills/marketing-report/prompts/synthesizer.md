## КРИТИЧЕСКИЕ ПРАВИЛА

### Формат

1. **ТОЛЬКО чистый Markdown.** Никакого HTML: никаких `<table>`, `<tr>`, `<td>`, `<callout>`, `<br>`. Notion API не рендерит HTML — он показывается как текст.
2. **Таблицы** — ТОЛЬКО pipe-формат: `| Колонка1 | Колонка2 |`
3. **Toggle-заголовки** — `## ▶ Название секции` (символ ▶ обязателен для свёртки в Notion)
4. **Форматирование** — `**жирный**`, `*курсив*`, `` `код` ``
5. **Разделители** — `---` между секциями
6. **ASCII-воронки** — ТОЛЬКО внутри блоков ` ``` ` (тройные обратные кавычки)

### Данные

7. **ТОЛЬКО реальные данные.** Если данных нет — писать "Данные за период отсутствуют" с пояснением. НИКОГДА не придумывать числа.
8. **ТОЛЬКО реальные модели** из входных данных. ЗАПРЕЩЕНО придумывать модели (Devi, Luna, Elsa, Ariel, Jasmine НЕ СУЩЕСТВУЮТ).
9. **ВСЕ ячейки заполнены.** Нет пустых `—` если данные есть во входных данных. Если данных реально нет — написать "н/д" с причиной.

### Терминология (РУССКАЯ)

10. ДРР (не DRR/Ad spend ratio), СПП (не SPP/discount), Выручка (не Revenue), Маржа (не Margin), Заказы (не Orders), Комиссия (не Commission), Логистика (не Logistics), Выкуп (не Redemption rate).

### Числа

11. Формат: `1 234 567 ₽`, `24,1%`, `+3,2 п.п.`
12. Крупные суммы: `8,8М` или `42,8 млн` — не `8 800 000 ₽` (читабельность)
13. Значимые изменения выделять bold: `**+137,7%**`, `**-2367%**`

### Стилистика (эталон: Маркетинговый анализ за 16-22 марта 2026)

14. **Читаемость** — короткие предложения, причинно-следственные цепочки, конкретные цифры
15. **Callout-блоки** — после ключевых секций (I, III, VI, IX, XI) добавлять callout с выводом:
    - Позитивный: `> ✅ Текст вывода`
    - Негативный: `> ❌ Текст вывода`
    - Предупреждение: `> ⚠️ Текст вывода`
    - Инсайт: `> 💡 Текст вывода`
    - Нейтральный/итоговый: `> 📊 Текст вывода`
16. **Подсветка в таблицах** — значимые Δ выделять **bold** прямо в ячейках таблиц (поддерживается)
17. **1-3 строки интерпретации** после каждой таблицы — что произошло и почему
18. **ДРР** — ВСЕГДА с разбивкой: внутренний (МП) и внешний (блогеры, ВК) отдельно
19. **Внешняя реклама** — двойной KPI: ДРР % + заказы через внешний трафик (шт и ₽). Без заказов → реклама неэффективна

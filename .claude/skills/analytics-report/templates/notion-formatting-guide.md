# Notion Formatting Guide — Analytics Reports

## КРИТИЧНО: ТОЛЬКО чистый Markdown

`shared/notion_client.py` → `md_to_notion_blocks()` конвертирует **чистый Markdown** в нативные Notion-блоки.

**HTML НЕ ПОДДЕРЖИВАЕТСЯ.** Теги `<table>`, `<tr>`, `<td>`, `<callout>` показываются как текст.

---

## Таблицы

ТОЛЬКО pipe-формат:

```markdown
| Показатель | Текущий, тыс.₽ | Прошлый, тыс.₽ | Δ | Δ% |
|---|---|---|---|---|
| Выручка | 36 482 | 31 205 | +5 277 | +16,9% |
| Маржа | 8 800 | 7 200 | +1 600 | +22,2% |
| **ИТОГО** | **43 077** | **37 800** | **+5 277** | **+14,0%** |
```

Рендерится как нативная Notion-таблица с заголовком.

## Toggle-заголовки (сворачиваемые секции)

Символ ▶ после ## создаёт toggle heading в Notion:

```markdown
## ▶ I. Паспорт отчёта

Контент внутри toggle...

| Параметр | Значение |
|---|---|
| Период | 30.03.2026 — 05.04.2026 |

## ▶ II. Следующая секция
```

Каждая секция сворачиваема. Читатель видит outline и раскрывает нужные.

**Уровни:**
- `## ▶` — основные секции (I-XIV)
- `### ▶` — подсекции (VII.1, VII.2...)
- Без ▶ — обычный заголовок (не сворачивается)

## Форматирование текста

| Markdown | Результат в Notion |
|---|---|
| `**жирный**` | **жирный** |
| `*курсив*` | *курсив* |
| `` `код` `` | `код` |
| `[текст](url)` | ссылка |
| `- пункт` | маркированный список |
| `1. пункт` | нумерованный список |
| `---` | горизонтальная линия |

## Формат чисел

| Тип | Формат | Пример |
|---|---|---|
| Деньги (рубли) | Пробел-разделитель + ₽ | `1 234 567 ₽` |
| Деньги (тысячи) | + тыс.₽ | `36 482 тыс.₽` |
| Проценты | Одна десятая | `24,1%` |
| Изменение % | Со знаком | `+16,9%` или `-3,2%` |
| Изменение п.п. | С указанием п.п. | `+3,2 п.п.` |
| Штуки | Пробел-разделитель | `1 234 шт` |
| Дни | + дн | `45 дн` |
| Отсутствие данных | Тире | `—` (не "-", не "N/A") |

## Формат дат

| Контекст | Формат | Пример |
|---|---|---|
| В тексте | Русский полный | `17 марта 2026` |
| В таблицах | ДД.ММ.ГГГГ | `17.03.2026` |
| Период | Через тире | `01.03 — 31.03.2026` |
| ЗАПРЕЩЁН | ISO | ~~2026-03-17~~ |

## Заголовки

- Русский язык, без аббревиатур (кроме WB, OZON, ДРР, СПП, ROI)
- Модели — с заглавной буквы: Wendy, Audrey, Ruby
- В GROUP BY: LOWER(), но в отображении Title Case

## Терминология (РУССКАЯ)

| Правильно | Неправильно |
|---|---|
| Себестоимость | COGS |
| Маржинальность | Margin% |
| Оборачиваемость | Turnover |
| Выручка | Revenue |
| Комиссия | Commission |
| Логистика | Logistics |
| Хранение | Storage |

## Структура секций

Секции нумеруются римскими цифрами: I, II, III... XIV.
Формат: `## ▶ I. Паспорт отчёта`
Между секциями — `---`

## Что НЕ поддерживается

- HTML теги любые (`<table>`, `<callout>`, `<tr color>`) — будут текстом
- Цвета строк/ячеек — Notion API поддерживает, но md_to_notion_blocks() нет
- Callout-блоки — нет синтаксиса в markdown
- Вложенные таблицы

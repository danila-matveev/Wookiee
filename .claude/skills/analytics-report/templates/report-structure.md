# Report Structure Template — Analytics Report

13 секций. Каждая обязательна. Пропуск секции = ошибка.

---

## Section 0: Executive Summary

**Toggle Heading 1**: `Резюме`

10-15 строк. Оформляется как callout block:

```html
<callout icon="📊" color="blue_bg">
  Краткое резюме всех ключевых находок.
  Период: {current_start} — {current_end} vs {prev_start} — {prev_end}.
  
  Ключевые цифры:
  - Выручка: X ₽ (Δ Y%)
  - Маржа: X ₽ / Z% (Δ Y п.п.)
  - Заказы: X шт (Δ Y%)
  - ДРР: X% (внутр. A% + внешн. B%)
  - Оборачиваемость: X дн
  
  Главный вывод в 1-2 предложениях.
</callout>
```

Синтезирует находки ВСЕХ секций. Пишется последним, после всех остальных секций.

## Section 1: Топ-находки

**Toggle Heading 1**: `Топ-находки`

3-5 ключевых находок. Каждая с эффектом в рублях:

```
1. {Описание находки} — эффект: {+/- X ₽/мес}
2. ...
```

Сортировка по абсолютному эффекту (от большего к меньшему). Каждая находка:
- Что произошло (факт с числами)
- Почему (причина, если HIGH/MEDIUM confidence)
- Эффект в ₽/мес

## Section 3: Ключевые изменения бренда

**Toggle Heading 1**: `Ключевые изменения бренда`

Таблица 15 строк — по одной на каждую активную модель. БЕЗ СОКРАЩЕНИЙ — все модели обязательны.

| Модель | Выручка тек., тыс.₽ | Выручка пред., тыс.₽ | Δ% | Маржа тек., тыс.₽ | Маржа пред., тыс.₽ | Δ% | Маржин-ть тек. | Маржин-ть пред. | Δ п.п. | Заказы, шт | Δ% | Статус |
|---|---|---|---|---|---|---|---|---|---|---|---|---|

Строка ИТОГО внизу (gray_bg). Модели отсортированы по Δ маржи (от лучшего к худшему).

Статус = цветная метка:
- Зеленый: маржинальность выросла на 2+ п.п.
- Красный: маржинальность упала на 2+ п.п.
- Желтый: изменение <2 п.п.

## Section 10: Гипотезы, действия, контрольные метрики

**Toggle Heading 1**: `Гипотезы → Действия → Контроль`

5-15 пунктов. Каждый пункт:

| # | Гипотеза | Confidence | Действие | Ожидаемый эффект, ₽/мес | Контрольная метрика | Срок проверки |
|---|---|---|---|---|---|---|

Сортировка: HIGH confidence сверху, затем MEDIUM, затем LOW.

Правила:
- HIGH: конкретное действие с цифрами
- MEDIUM: действие с оговоркой "при подтверждении"
- LOW: только наблюдение

## Section 11: Рекомендации советника

**Toggle Heading 1**: `Рекомендации советника`

Цветовая маркировка приоритета:

- 🔴 **Срочно** (действие в течение 1-3 дней)
- 🟡 **Важно** (действие в течение недели)
- 🟢 **Рекомендация** (действие в течение месяца)

## Общие правила по секциям

1. Каждая секция — toggle heading (сворачиваемый)
2. Каждая таблица — Notion-native `<table>` (не markdown)
3. Каждое число — из data_layer, не выдуманное
4. Каждое сравнение — current vs previous, оба периода рассчитаны
5. Каждая рекомендация — с эффектом в ₽
6. Модели — ВСЕГДА с заглавной буквы (Wendy, Devi, Luna)
7. Все заголовки — на русском языке
8. Формат чисел: `1 234 567 ₽`, `24.1%`, `+3.2 п.п.`
9. Формат дат: "17 марта 2026" в тексте, "17.03.2026" в таблицах

## Запрещенные формулировки

НИКОГДА не использовать:

| ЗАПРЕЩЕНО | ПРАВИЛЬНО |
|---|---|
| "Внешняя реклама прекратилась" | "Сегодня без внешней рекламы" |
| "Внешняя реклама выросла в X раз" (сравнение дня с блогером vs без) | "Размещение у блогера X руб" |
| "Экономия на внешней рекламе +X руб" | Нельзя считать отсутствие несистемного расхода экономией |
| "ДРР внешний 45% в день блогера" | "Размещение X руб. ДРР считать рано (лаг 3-7 дн.)" |
| "Реклама у блогера неэффективна" (если < 7 дней) | "Результаты размещения ожидаются через 3-7 дней" |

---

## Жесткие правила

1. **Блогеры — случайный канал.** Дневной анализ ЗАПРЕЩЕН. Только за период.
2. **ВК/Яндекс — систематический канал.** Дневной анализ РАЗРЕШЕН.
3. **ДРР внешний** — показывать отдельно от внутреннего. Никогда не смешивать.
4. **ROI блогеров** — считать с лагом 3-7 дней. Если размещение свежее — пометить.
5. **Формат чисел:** `1 234 567 руб`, `24.1%`.
6. **Рекомендации** — конкретные: "увеличить бюджет ВК на X руб", "повторить размещение у блогера Y".
7. **Тон** — сухой, фактологический.

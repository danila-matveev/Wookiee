# Задачи

## 1. ДРР по артикулам
- ДРР = (внутренняя + внешняя реклама) / выручка × 100
- Разбивка: внутренняя (МП) и внешняя отдельно
- Переинвестированные: ДРР >15%
- Недоинвестированные: ДРР <3% при хорошей марже (A/B категория)

### Помесячная динамика ДРР
По каждому артикулу с аномальным ДРР — показать ДРР за m1, m2, m3 отдельно:
- Как менялся ДРР? Рос, падал, стабилен?
- В каком месяце была максимальная реклама? Какой результат (заказы, маржа)?
- Разбивка внутренняя/внешняя по каждому месяцу — что использовалось?

### Проверка ранее протестированных гипотез
Перед рекомендацией "увеличить ДРР":
- Проверить: был ли месяц с бОльшими расходами? Какой результат?
- Если гипотеза уже тестировалась и не сработала — НЕ рекомендовать повторно
- Формат: "В m2 ДРР был 15% (vs m3 = 5%), заказы +20%, маржа -5% → реклама неэффективна по марже"

## 2. Связка реклама → заказы
- ДРР растёт — растут ли заказы (шт и руб)?
- Реклама + рост заказов = эффективно
- Реклама растёт, заказы нет = неэффективно → флаг

## 3. Конверсионные аномалии
- Если есть трафик (высокие orders_count), но низкая конверсия в выкуп → проблема карточки/контента
- Если ДРР высокий, а продажи низкие → неэффективная реклама

## 4. Перераспределение маркетинг-бюджета
- Снять рекламу с C-товаров (низкая маржа → ДРР бессмысленный)
- Добавить на A-товары с потенциалом роста
- Конкретные рекомендации: "перенести X₽ с C-товара Y на A-товар Z"

## 5. Недоинвестированные звёзды
- A-товары с ДРР <3% — можно ли ускорить продажи рекламой?
- Потенциал в руб/мес при увеличении ДРР до 5-8%

## 6. Анализ ДРР на уровне склеек

Данные о склейках: `hierarchy.skleyka_groups` (из enriched data).

Для каждой склейки с >1 артикулом:
- Суммарная выручка и маржа склейки
- Суммарные рекламные расходы склейки
- Групповой ДРР = сумма рекламы / сумма выручки × 100
- Какие артикулы рекламируются? Какие нет?
- Эффект: если один артикул рекламируется, а продажи растут у ДРУГИХ артикулов в склейке — реклама работает на группу

### Перед снижением рекламы
Проверить: не является ли рекламируемый артикул трафик-магнитом для всей склейки?

### Недоинвестированные
Если артикул без рекламы, но в его склейке есть рекламируемый артикул — он УЖЕ получает косвенный трафик. Рекомендация "добавить рекламу" может быть избыточной.

# Правила
- Причинно-следственные цепочки обязательны
- ТОЛЬКО реальные данные
- ДРР — ВСЕГДА с разбивкой внутренняя/внешняя
- Реклама → заказы: всегда анализировать связку
- Не рекомендовать убирать рекламу, если заказы растут
- Никогда не рекомендовать "остановить всю рекламу" — только "протестировать приостановку/снижение на 2 недели"
- Реклама на убыточный артикул может генерировать органический трафик на другие цвета
- ДРР анализировать НЕ ТОЛЬКО по артикулу, но и на уровне склейки (групповой ДРР)
- Перед "снизить рекламу" — проверить, не является ли артикул трафик-магнитом для склейки

# Правила форматирования

## Терминология — только русский
| Запрещено | Правильно |
|---|---|
| COGS | Себестоимость |
| Margin% | Маржинальность |
| Turnover | Оборачиваемость |
| Revenue | Выручка |
| ROI | Годовая доходность (ROI) |
| DRR | ДРР |

## Числа
| Тип | Формат | Пример |
|---|---|---|
| Рубли | Пробел-разделитель + ₽ | 1 234 567 ₽ |
| Крупные суммы | Сокращённо | 8,8М ₽ |
| Проценты | Одна десятая | 24,1% |
| Изменение % | Со знаком | +16,9%, -3,2% |
| п.п. | п.п. | +3,2 п.п. |
| Штуки | Пробел | 1 234 шт |
| Дни | + дн | 45 дн |
| Нет данных | — | — |

Значимые изменения (>+/-10%) — **жирным**.

## Причинно-следственные цепочки
НЕ: "Joy/чёрный — кандидат на вывод"
ДА: "Joy/чёрный: маржинальность 18%, ROI 72% (vs медиана 176%). ДРР = 2% → недоинвестирован → низкие продажи (2 шт/д vs медиана 8) → медленная оборачиваемость (92 дн)"

# Правила
- ТОЛЬКО данные из вердиктов Арбитра. Никакой собственной аналитики.
- Все ячейки заполнены если данные есть
- Нет данных → "—"
- Никогда не выдумывать модели или артикулы
- Если причина проблемы неизвестна — "требуется проверка", НЕ предполагать конкретную причину
- Недопустимо: "фото не соответствуют цвету". Допустимо: "требуется аудит карточки"
- "Возврат из вывода" — всегда с оговоркой "требует подтверждения причины вывода"
- Рекомендации по рекламе — "протестировать", не "остановить"

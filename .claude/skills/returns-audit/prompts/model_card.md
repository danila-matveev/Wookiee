
### Контекст из Wave 1:
- Тренд-анализ: {{TREND_FINDINGS}}
- Причины (кластеры): {{REASON_FINDINGS}}

## Задачи

### A. Обзор модели
- Кол-во заявок на возврат
- % к продажам (orders)
- Сравнение с средним rate по бренду

### B. Причины возвратов этой модели
- Распределение по причинам (из кластеров Wave 1)
- Уникальные причины (только у этой модели)

### C. Хронология
- Распределение заявок по дням — есть ли всплески?
- Связь с конкретными поставками или акциями?

### D. Комментарии покупателей
- 3-5 наиболее информативных комментариев (дословно)
- Общий тон: негативный / нейтральный

### E. Рекомендация
- Конкретное действие для снижения возвратов этой модели
- Приоритет: HIGH / MEDIUM / LOW

# Pull Request Review Skill

## Описание
Автоматический workflow создания и проверки Pull Request.

## Workflow

### 1. Самопроверка
- Прочитай AGENTS.md
- Проверь DoD (agent_docs/guides/dod.md)
- Убедись что нет секретов в трекаемых файлах

### 2. Подготовка
- Создай feature branch если на main
- Закоммить все изменения
- Обнови agent_docs/development-history.md

### 3. Создание PR
- Создай PR через `gh pr create`
- Формат описания:
  ```
  ## Summary
  - Что сделано
  - Зачем

  ## Test plan
  - Как проверить
  ```

### 4. Проверка
- Код работает по требованиям
- Нет регрессий
- Документация актуальна
- Нет секретов в diff

### 5. Завершение
- Режим по умолчанию: auto-merge после проверки
- Режим "wait": спросить пользователя

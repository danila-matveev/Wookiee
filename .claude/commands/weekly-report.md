Создай еженедельный аналитический отчёт. $ARGUMENTS — дата любого дня недели (по умолчанию — прошлая неделя).

Workflow Ромы (ИИ финн-аналитик Wookiee):

1. Вычисли понедельник и воскресенье для указанной недели:
   - Если $ARGUMENTS пустой — возьми прошлую неделю (пн—вс)
   - Если указана дата — найди пн и вс той недели

2. Запусти скрипт:
   ```
   python3 scripts/period_analytics.py --start {понедельник} --end {воскресенье} --save --export-context
   ```

3. Прочитай справочник Ромы: `scripts/analytics_agent_roma/playbook.md`

4. Прочитай данные для анализа: `reports/{понедельник}_{воскресенье}_data_context.json`

5. На основе playbook (разделы 6 и 7.1) и data_context напиши **два формата** анализа:

   **А. Краткая сводка (для чата)** — по формату из playbook раздел 6.1:
   - Максимум 30-40 строк, форматирование [b]...[/b]
   - «Сводка за неделю {дата—дата}:»
   - Бренд → WB → OZON → Декомпозиция → Что произошло → Рекомендации
   - Контрфактуал: «если убрать X → маржа была бы Y руб (Z%)»

   **Б. Подробный отчёт (для Notion)** — по формату из playbook раздел 6.2 + 7.1:
   - Причинно-следственные цепочки с конкретными цифрами
   - Дневная динамика внутри недели (лучший/худший день)
   - Модельная декомпозиция (таблица)
   - Связка реклама → заказы на недельном уровне
   - Рекомендации с оценкой эффекта в рублях

6. Допиши **подробный отчёт** в файл `reports/{понедельник}_{воскресенье}_analytics.md`

7. Синхронизируй с Notion:
   ```
   python3 scripts/notion_sync.py --file reports/{понедельник}_{воскресенье}_analytics.md
   ```

8. Покажи пользователю **краткую сводку** (готовую для копирования в чат) и ссылку на Notion

Создай аналитический отчёт за произвольный период. $ARGUMENTS — две даты через пробел: начало и конец периода.

Пример: /period-report 2026-02-01 2026-02-07

Workflow Ромы (ИИ финн-аналитик Wookiee):

1. Разбери даты из $ARGUMENTS (формат: YYYY-MM-DD YYYY-MM-DD)

2. Запусти скрипт:
   ```
   python3 scripts/period_analytics.py --start {начало} --end {конец} --save --export-context
   ```

3. Прочитай справочник Ромы: `scripts/analytics_agent_roma/playbook.md`

4. Прочитай данные для анализа: `reports/{начало}_{конец}_data_context.json`

5. На основе playbook (разделы 6 и 7.3) и data_context напиши **два формата** анализа:

   **А. Краткая сводка (для чата)** — по формату из playbook раздел 6.1:
   - Максимум 30-40 строк, форматирование [b]...[/b]
   - «Сводка за {начало}—{конец}:»
   - Бренд → WB → OZON → Маркетинг → Декомпозиция → Что произошло → Рекомендации
   - Адаптируй формат под длину периода (см. раздел 7.3)
   - Контрфактуал: «если убрать X → маржа была бы Y руб (Z%)»

   **Б. Подробный отчёт (для Notion)** — по формату из playbook раздел 6.2:
   - Причинно-следственные цепочки с конкретными цифрами
   - Модельная декомпозиция (таблица)
   - **Маркетинг и воронка:** рекламная эффективность по моделям (CPM, CTR, CPO, ROMI-статус), органическая воронка (конверсии по шагам), связка реклама ↔ органика, гипотезы
   - Рекомендации с оценкой эффекта в рублях

6. Допиши **подробный отчёт** в файл `reports/{начало}_{конец}_analytics.md`

7. Синхронизируй с Notion:
   ```
   python3 scripts/notion_sync.py --file reports/{начало}_{конец}_analytics.md
   ```

8. Покажи пользователю **краткую сводку** (готовую для копирования в чат) и ссылку на Notion

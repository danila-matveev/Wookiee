Работа с Google Sheets через gws CLI.
$ARGUMENTS — описание задачи (напр. "прочитай данные из таблицы 1BxiMVs0XRA5nFMdKvBdBZjgmUUqptlbs74OgVE2upms диапазон A1:D10").

Используй скилл gws-sheets для выполнения задачи.
Перед write-операциями (append, update, clear) обязательно подтверди у пользователя.

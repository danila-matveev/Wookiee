# scripts/logistics_report/

Implementation модуль для скилла `/logistics-report`.

Анализ логистики WB+OZON — расходы, индекс локализации, возвраты, остатки, оборачиваемость, рекомендации по допоставкам. Еженедельный + ежемесячный.

## Содержимое
- `collect_data.py` — сбор данных логистики из БД и WB Logistics API

# scripts/funnel_report/

Implementation модуль для скилла `/funnel-report`.

Глубокая воронка по WB — per-model funnel (переходы → корзина → заказы → выкупы), CRO как главная метрика, CR на каждом шаге с Δ п.п., экономика, значимые артикулы, рекомендации с эффектом в ₽.

## Содержимое
- `validate_output.py` — валидатор выхода отчёта (data quality checks)

# data/branding — фирменные аватарки Саймона

Оператор кладёт PNG сюда после merge T1, используя указанные имена файлов.

## Файлы

| Файл | Назначение | Размер | Где используется |
|------|-----------|--------|------------------|
| `saimon_avatar_square.png` | Квадратная аватарка | 640×640 px | Telegram (@BotFather → /setuserpic) |
| `saimon_avatar_telemost.png` | Прямоугольная аватарка | 200×200 px (или native Yandex 360) | admin.yandex.ru → recorder@wookiee.shop |

## Инструкция для оператора

1. Подготовь квадратный PNG 640×640 и назови его `saimon_avatar_square.png`.
2. Подготовь прямоугольный PNG для Yandex 360 профиля и назови его `saimon_avatar_telemost.png`.
3. Положи оба файла в эту папку и закоммить.
4. Загрузи `saimon_avatar_square.png` через @BotFather → /setuserpic.
5. Загрузи `saimon_avatar_telemost.png` через admin.yandex.ru → профиль recorder@wookiee.shop.

# wb-prices — WB Цены

## Назначение

Выгружает актуальные цены и скидки на товары из Wildberries по всем кабинетам. Используется как основной справочник цен, а также как источник данных для листа Склейки WB.

---

## API источник

| Параметр | Значение |
|---|---|
| Host | `discounts-prices-api.wildberries.ru` |
| Endpoint | `GET /api/v2/list/goods/filter` |
| Auth | `Authorization: Bearer {WB_API_KEY}` |

### Параметры запроса

| Параметр | Значение |
|---|---|
| `limit` | `1000` (максимум на страницу) |
| `offset` | `0`, `1000`, `2000`, ... (пагинация) |

### Формат ответа

```json
{
  "data": {
    "listGoods": [
      {
        "nmID": 123456789,
        "vendorCode": "ART-001",
        "sizes": [
          {
            "price": 2500,
            "discountedPrice": 2000,
            "discount": 20
          }
        ]
      }
    ]
  }
}
```

Цена берётся из первого элемента массива `sizes`.

---

## Целевой лист

- **Название листа:** `WB Цены`
- **Чекбокс запуска:** ячейка `C1` — скрипт стартует только если значение `true`, после завершения сбрасывает в `false`.

---

## Колонки

| № | Заголовок | Источник поля |
|---|---|---|
| 1 | nmID | `nmID` |
| 2 | Артикул | `vendorCode` |
| 3 | Кабинет | Имя кабинета (ООО / ИП) |
| 4 | Цена | `sizes[0].price` |
| 5 | Скидка% | `sizes[0].discount` |

---

## Логика обработки

```
для каждого кабинета (ООО, ИП):
    offset = 0
    цикл:
        GET /api/v2/list/goods/filter?limit=1000&offset={offset}
        извлечь listGoods
        если listGoods пуст — выйти из цикла
        добавить записи в общий список с полем "Кабинет"
        offset += 1000

записать заголовок + все строки на лист "WB Цены"
очистить чекбокс C1
```

---

## Обработка ошибок

| Ситуация | Поведение |
|---|---|
| HTTP 401 | Логировать "Неверный токен кабинета X", пропустить кабинет |
| HTTP 429 (rate limit) | Пауза 60 секунд, повторить запрос |
| Пустой `listGoods` в первом ответе | Записать пустой лист (только заголовок) |
| Сетевая ошибка | До 3 повторных попыток с экспоненциальной паузой |
| Отсутствует поле `sizes` | Пропустить запись, залогировать предупреждение |

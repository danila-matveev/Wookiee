# moysklad — МойСклад

## Назначение

Выгружает полный товарный каталог из МойСклад: артикулы, баркоды, атрибуты номенклатуры, остатки по складам (основной, приёмка, в пути) и себестоимость из последних закупочных заказов. Является основным справочником товаров в системе.

---

## API источник

| Параметр | Значение |
|---|---|
| Host | `api.moysklad.ru` |
| Base path | `/api/remap/1.2` |
| Auth | `Authorization: Bearer {MOYSKLAD_TOKEN}` |
| Content-Type | `application/json;charset=utf-8` |

### Используемые эндпоинты

| Эндпоинт | Метод | Назначение |
|---|---|---|
| `/entity/assortment` | GET | Каталог товаров (с пагинацией) |
| `/report/stock/all` | GET | Остатки по складам |
| `/entity/purchaseorder` | GET | Закупочные заказы (для себестоимости) |

### Параметры запроса `/entity/assortment`

```
limit=500
offset=0, 500, 1000, ...
expand=product.attributes
```

## Целевой лист

- **Название листа:** `МойСклад_АПИ`
- **Чекбокс запуска:** ячейка `C1` — скрипт стартует только если значение `true`, после завершения сбрасывает в `false`.

## Логика обработки

```
# Шаг 1 — Загрузка каталога
offset = 0
цикл:
    GET /entity/assortment?limit=500&offset={offset}&expand=product.attributes
    если rows пуст — выйти
    отфильтровать: только записи с атрибутами И более одного баркода
    извлечь 32 атрибута в словарь
    offset += 500

# Шаг 2 — Остатки
GET /report/stock/all
для каждой записи: сопоставить по ID товара → заполнить AM, AN, AO

# Шаг 3 — Себестоимость
GET /entity/purchaseorder (последние заказы, сортировка по дате DESC)
для каждого товара: найти последнюю закупочную цену → заполнить AP

# Шаг 4 — Запись
записать заголовки + все строки на лист "МойСклад_АПИ"
очистить чекбокс C1
```

## Обработка ошибок

| Ситуация | Поведение |
|---|---|
| HTTP 401 | Бросить исключение `AuthError`, завершить выполнение |
| HTTP 429 (rate limit) | Пауза 60 секунд, повторить запрос |
| Отсутствуют атрибуты у товара | Пропустить запись (не включать в выгрузку) |
| Баркодов менее 2 | Пропустить запись |
| Товар не найден в остатках | Записать 0 в колонки AM–AO |
| Нет закупочных заказов | Записать пустую строку в AP |
| Сетевая ошибка | До 3 повторных попыток с экспоненциальной паузой |

# wb_localization / calculators

Sub-пакет родительского модуля `services/wb_localization/`. Содержит pure-Python калькуляторы (без I/O), которые принимают сырые данные WB/МойСклад и возвращают аналитические структуры. Импортируются из `run_localization.py` и `generate_localization_report_v3.py`.

## Файлы
- `il_irp_analyzer.py` — per-article ИЛ и ИРП по 6 ФО, ранжирование по вкладу `(КТР-1)×заказы`
- `economic_analyzer.py` — legacy 3-сценарийный анализ («Без контроля» / «Сейчас» / «Оптимизированный»)
- `scenario_engine.py` — новый сценарный анализ при 30–90% локализации (с шагом 10%) на реальных артикулах и ценах
- `relocation_forecaster.py` — симуляция понедельного эффекта перестановок на 13 недель (скользящее окно ИЛ)
- `reference_builder.py` — структура данных для листа «Справочник» (документация КТР/КРП, лимиты перераспределения)

См. родительский [README](../README.md) для общей картины пайплайна.
